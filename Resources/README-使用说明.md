# Nihongo ToastFish 使用说明

Nihongo ToastFish 是一个常驻 Windows 托盘的日语学习工具。主要学习方式是通过系统通知弹窗显示词汇、语法和例句，让你在使用电脑时低打扰地复习。

## 1. 快速开始

1. 启动软件后，在右下角托盘找到 Nihongo ToastFish 图标。
2. 右键托盘图标，选择“继续上次学习”，会从上次退出或暂停的位置继续。
3. 如果是第一次使用，可以选择：
   - “日语学习词库” -> “N5 词汇”
   - “日语学习词库” -> “N5 语法”
   - “日语学习词库” -> “N5 例句”
4. 弹窗出现后，可以用按钮或快捷键切换上一条、下一条、添加笔记或暂停。

## 2. 日语学习内容

内置内容按 JLPT 参考等级组织，包含 N5 到 N1：

- 词汇：日语词条、读音、中文释义、词性和例句。
- 语法：语法形式、中文释义、接续说明和用法说明。
- 例句：围绕语法和表达的例句练习。
- 五十音：假名顺序学习和练习。

内置词库可以直接使用，不需要先导入 Excel。Excel 导入更适合自定义学习内容。

## 3. 弹窗按钮

日语学习弹窗通常包含这些按钮：

- 上一个：返回上一条内容。
- 下一个：进入下一条内容。
- 添加笔记：把当前词汇、语法或例句加入学习笔记本。
- 暂停：停止当前学习流程。

点击弹窗右上角关闭按钮会暂停当前学习，不会继续自动弹出下一条。

## 4. 学习笔记本

在托盘菜单中点击“笔记本”可以打开学习笔记本。

笔记会自动分类：

- 词汇笔记：保存词条、读音、释义和例句。
- 语法笔记：保存语法形式、释义和说明。
- 例句笔记：保存提示、例句和答案。

如果同一条内容已经添加过，再次添加不会重复保存。

## 5. 练习模式

托盘菜单中的“练习”包含：

- 日语单词练习
- 日语语法练习
- 五十音练习
- 英语单词练习

日语练习弹窗支持“上一个”和“下一个”。点击右上角关闭按钮会暂停练习。

## 6. 快捷键

常用全局快捷键如下：

| 快捷键 | 功能 |
| --- | --- |
| Alt+Q | 继续上次学习 |
| Alt+` | 播放当前发音 |
| Alt+1 到 Alt+4 | 对应点击通知按钮 1 到 4 |
| Alt+A | 上一个 |
| Alt+D | 下一个 |
| Alt+S | 添加当前内容到笔记本 |
| Alt+E | 打开当前内容详情 |
| Alt+W | 关闭当前弹窗并暂停 |
| Ctrl+Alt+J | 继续上次学习 |
| Ctrl+Alt+V | 从 N5 词汇开始 |
| Ctrl+Alt+G | 从 N5 语法开始 |
| Ctrl+Alt+E | 从 N5 例句开始 |
| Ctrl+Alt+P | 暂停当前学习 |
| Ctrl+Alt+O | 显示或隐藏主窗口 |

如果某个快捷键没有反应，通常是被其他软件占用。可以先关闭占用快捷键的软件后再试。

## 7. 常用菜单说明

| 菜单 | 说明 |
| --- | --- |
| 继续上次学习 | 从上次学习位置继续 |
| 开始学习 | 使用当前选择的词库开始学习 |
| 日语学习词库 | 选择内置日语词汇、语法、例句和五十音 |
| 练习 | 进入单词、语法或五十音练习 |
| 笔记本 | 查看已收藏的重点词汇、语法和例句 |
| 导入自定义内容 | 导入自己的 Excel 词库 |
| 设置 | 设置每组数量、自动发音、日志和进度 |
| 帮助 | 查看快捷键、使用说明和项目链接 |
| 退出 | 退出软件 |

## 8. 自定义内容导入

如果你有自己的学习内容，可以通过“导入自定义内容”导入 Excel。

建议优先使用内置日语词库熟悉流程，再导入自定义内容。导入失败时，通常是 Excel 格式和模板不一致，可以参考软件自带模板重新整理。

## 9. 通知和语音

- 学习内容通过 Windows 通知显示。
- 弹窗显示时间已设置为较长模式，但仍会受到 Windows 通知设置影响。
- 如果日语发音不可用，请在 Windows 设置中安装日语语音包。
- 如果完全看不到弹窗，请检查 Windows 通知设置是否允许 Nihongo ToastFish 发送通知。

## 10. 学习建议

- 初学者建议从 N5 词汇开始，然后加入 N5 语法。
- 遇到重点内容时用“添加笔记”保存，不要在弹窗里停留太久。
- 每天少量多次使用“继续上次学习”，比一次性刷很多条更稳定。
- 复习时可以先看笔记本，再继续新内容。
